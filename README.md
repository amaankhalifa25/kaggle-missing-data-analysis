# Data Cleaning Challenge – Python

## Overview

This repository contains a collection of data-cleaning exercises completed using Python and pandas, based on Kaggle’s “5-Day Data Cleaning Challenge”. 
This Challenge is available on the kaggle website via this link: https://www.kaggle.com/code/rtatman/data-cleaning-challenge-handling-missing-values

The project focuses on preprocessing and cleaning techniques used in real-world data science scenarios. Each notebook demonstrates practical methods for identifying, cleaning, transforming, and preparing messy datasets for analysis.

The repository was completed as part of independent practice to strengthen foundational data science and data preprocessing skills.
